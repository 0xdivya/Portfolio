CallShield – Fraud Call Detection App

Technologies Used: Flutter, Firebase, Dart, NLP, REST API

CallShield is a real-time fraud detection mobile application designed to protect users from scam calls involving OTPs, banking threats, and fake emergency requests.

  • Real-time call monitoring and analysis
  • NLP-based detection of fraud patterns
  • Instant alert system for user safety
  • Cloud data handling using Firebase

This project strengthened my understanding of mobile security and real-time data processing concepts.
